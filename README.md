# Python Daily Wins

## Introdução

Este repositório é meu espaço seguro para programar no meu ritmo e sem julgamentos. Antes de iniciar este projeto, passei meses sem programar, havia perdido o prazer nesta atividade, de programar para mim, e de gostar desta atividade. Retomo esta atividade através deste repositório, com o intuito de resgatar meu prazer e minha confiança na programação. Assim, como aprendi na minha terapia, o mantra pricipal deste repositório é:

" Pouco, pequeno e devagar"

## Por que resolver problemas do Codewars e similares?

Resolver problema do codewars me parece o melhor e único caminho. Lembro-me que na faculdade eu adorava resolver problemas de programação do site URI (hoje deve se chamar bee alguma coisa). Sou uma pessoa competitiva, perfeccionista e adoro lógica de programação e matemática. Logo, estes tipos de site tem a receita perfeita para me desafiar de forma gradual (há um certo grau de gameficação nos níveis dos problemas e isso é muito bom). Há anos que não resolvo problemas destes sites, e há ainda problemas que nunca consegui resolver na época e que eu adoraria saber resolver no futuro. Retomar aos poucos, do problemas mais fáceis aos mais difíceis, me parece ser o caminho para construir minha autoconfiança, retomar o prazer de resolver problemas de programação usando Python e construir o hábito de programar regularmente.

## Lembrete para mim mesmo

Caso, em alguma semana ou semanas eu não programe, está tudo bem. Não haverá julgamentos. Não haverá cobranças. Não haverá datas a cumprir. Este repositório estará aqui esperando o retorno. Caso eu não retorne, seja porque não faça mais sentido, ou porque consegui masterizar tudo (embora eu acredite que ao parar de resolver problemas de programação competitiva, eu fique um pouco mais enferrujado), tudo bem. Este é um lugar seguro, sempre que quiser voltar, serei bem-vindo.
